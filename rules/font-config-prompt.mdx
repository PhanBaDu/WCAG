# Font Configuration Prompt
## Cổng thông tin việc làm người khuyết tật — WCAG 2.2 AA

---

## NHIỆM VỤ

Cấu hình toàn bộ hệ thống font cho project **Next.js 14 + Tailwind CSS**.
Mục tiêu: đạt chuẩn **WCAG 2.2 Level AA** cho người khiếm thị (low vision + screen reader).

---

## BƯỚC 1 — Cài đặt font

Chạy lệnh sau:

```bash
npm install @fontsource/inter @fontsource/lexend
```

---

## BƯỚC 2 — Cấu hình Next.js (`app/layout.tsx`)

Thay thế hoàn toàn nội dung font hiện tại bằng đoạn sau:

```tsx
import type { Metadata } from 'next'
import { Inter, Lexend } from 'next/font/google'
import './globals.css'

const inter = Inter({
  subsets: ['latin', 'vietnamese'],
  variable: '--font-inter',
  display: 'swap',
  weight: ['400', '500', '600', '700'],
  preload: true,
  adjustFontFallback: true,
})

const lexend = Lexend({
  subsets: ['latin', 'vietnamese'],
  variable: '--font-lexend',
  display: 'swap',
  weight: ['400', '500', '600', '700'],
  preload: false,
  adjustFontFallback: true,
})

export const metadata: Metadata = {
  title: {
    template: '%s | Cổng Việc Làm Người Khuyết Tật',
    default: 'Cổng Việc Làm Người Khuyết Tật',
  },
  description: 'Nền tảng kết nối việc làm cho người khuyết tật đạt chuẩn WCAG 2.2',
}

export default function RootLayout({
  children,
}: {
  children: React.ReactNode
}) {
  return (
    <html
      lang="vi"
      className={`${inter.variable} ${lexend.variable}`}
      suppressHydrationWarning
    >
      <body className="font-sans antialiased">
        {children}
      </body>
    </html>
  )
}
```

### Rules bắt buộc:
- `lang="vi"` — bắt buộc theo WCAG 3.1.1, giúp screen reader đọc đúng tiếng Việt
- `display: 'swap'` — tránh FOIT (Flash of Invisible Text), người dùng thấy chữ ngay
- `subsets: ['vietnamese']` — load đúng bộ dấu tiếng Việt, không bị thiếu ký tự
- `adjustFontFallback: true` — giảm layout shift (CLS) khi font load xong
- `antialiased` — làm mịn nét chữ trên màn hình retina

## BƯỚC 3 — Cấu hình Tailwind (`tailwind.config.ts`)

Thay thế hoàn toàn phần `theme.extend` bằng đoạn sau:

```ts
import type { Config } from 'tailwindcss'

const config: Config = {
  content: [
    './pages/**/*.{js,ts,jsx,tsx,mdx}',
    './components/**/*.{js,ts,jsx,tsx,mdx}',
    './app/**/*.{js,ts,jsx,tsx,mdx}',
  ],
  theme: {
    extend: {

      // ── FONT FAMILY ──────────────────────────────────────────
      fontFamily: {
        // Primary: Inter — rõ ràng, tương phản cao, hỗ trợ tiếng Việt
        sans: [
          'var(--font-inter)',
          'Lexend',
          'ui-sans-serif',
          'system-ui',
          '-apple-system',
          'BlinkMacSystemFont',
          'Segoe UI',
          'sans-serif',
        ],
        // Fallback: Lexend — thiết kế cho low vision và khó đọc
        lexend: [
          'var(--font-lexend)',
          'var(--font-inter)',
          'sans-serif',
        ],
        // Monospace: dùng cho code, input OTP, CAPTCHA alternative
        mono: [
          'ui-monospace',
          'SFMono-Regular',
          'Menlo',
          'Consolas',
          'monospace',
        ],
      },

      // ── FONT SIZE ─────────────────────────────────────────────
      // Tất cả dùng rem để scale theo browser zoom của người dùng
      // WCAG 1.4.4: phải dùng được khi zoom 200%
      fontSize: {
        // Body text — tối thiểu 16px theo WCAG best practice
        'xs':   ['0.75rem',  { lineHeight: '1.5',  letterSpacing: '0.01em' }],
        'sm':   ['0.875rem', { lineHeight: '1.5',  letterSpacing: '0.01em' }],
        'base': ['1rem',     { lineHeight: '1.6',  letterSpacing: '0.01em' }],
        'lg':   ['1.125rem', { lineHeight: '1.6',  letterSpacing: '0.01em' }],
        'xl':   ['1.25rem',  { lineHeight: '1.5',  letterSpacing: '0.005em' }],
        // Heading
        '2xl':  ['1.5rem',   { lineHeight: '1.4',  letterSpacing: '-0.01em' }],
        '3xl':  ['1.875rem', { lineHeight: '1.35', letterSpacing: '-0.01em' }],
        '4xl':  ['2.25rem',  { lineHeight: '1.3',  letterSpacing: '-0.02em' }],
        '5xl':  ['3rem',     { lineHeight: '1.2',  letterSpacing: '-0.02em' }],
      },

      // ── FONT WEIGHT ───────────────────────────────────────────
      fontWeight: {
        normal:   '400',  // Body text
        medium:   '500',  // Label, caption
        semibold: '600',  // Button, sub-heading
        bold:     '700',  // Heading, emphasis
        // KHÔNG dùng 300 (light) — quá mỏng, kém tương phản cho low vision
        // KHÔNG dùng 800/900 — gây rối cho screen reader
      },

      // ── LINE HEIGHT ───────────────────────────────────────────
      // WCAG 1.4.12: line-height tối thiểu 1.5x font-size
      lineHeight: {
        none:     '1',
        tight:    '1.25',
        snug:     '1.375',
        normal:   '1.5',   // Tối thiểu cho body text
        relaxed:  '1.625', // Khuyến nghị cho paragraph
        loose:    '1.75',  // Dùng cho nội dung đặc biệt (help text)
        // Đừng dùng giá trị thấp hơn 1.5 cho paragraph text
      },

      // ── LETTER SPACING ────────────────────────────────────────
      // WCAG 1.4.12: letter-spacing tối thiểu 0.12x font-size
      letterSpacing: {
        tighter: '-0.02em',
        tight:   '-0.01em',
        normal:   '0em',
        wide:     '0.01em',  // Body text
        wider:    '0.02em',  // Small caps, label
        widest:   '0.05em',  // Không dùng cho đoạn dài
      },

      // ── MAX WIDTH CHO ĐOẠN VĂN ────────────────────────────────
      // WCAG best practice: 45–75 ký tự mỗi dòng
      maxWidth: {
        'prose-sm': '45ch',
        'prose':    '65ch',  // Dùng cho đoạn văn chính
        'prose-lg': '75ch',
      },

    },
  },
  plugins: [],
}

export default config
```

## BƯỚC 4 — CSS Global (`app/globals.css`)

Thêm các rule sau vào `globals.css`:

```css
@tailwind base;
@tailwind components;
@tailwind utilities;

/* ── BASE TYPOGRAPHY ─────────────────────────────────── */
@layer base {

  /* Root: dùng rem để tôn trọng font-size do người dùng chọn trong trình duyệt */
  html {
    font-size: 100%; /* = 16px mặc định, scale theo browser setting */
    -webkit-text-size-adjust: 100%;
    text-size-adjust: 100%;
    scroll-behavior: smooth;
  }

  /* Body */
  body {
    font-family: var(--font-inter), system-ui, sans-serif;
    font-size: 1rem;
    line-height: 1.6;
    letter-spacing: 0.01em;
    font-weight: 400;
    /* Tránh text quá nhỏ khi zoom */
    min-height: 100vh;
  }

  /* Heading hierarchy — WCAG 1.3.1 */
  h1 {
    font-size: 2.25rem;   /* 36px */
    font-weight: 700;
    line-height: 1.3;
    letter-spacing: -0.02em;
  }

  h2 {
    font-size: 1.875rem;  /* 30px */
    font-weight: 700;
    line-height: 1.35;
    letter-spacing: -0.01em;
  }

  h3 {
    font-size: 1.5rem;    /* 24px */
    font-weight: 600;
    line-height: 1.4;
    letter-spacing: -0.01em;
  }

  h4 {
    font-size: 1.25rem;   /* 20px */
    font-weight: 600;
    line-height: 1.5;
  }

  h5 {
    font-size: 1.125rem;  /* 18px */
    font-weight: 600;
    line-height: 1.5;
  }

  h6 {
    font-size: 1rem;      /* 16px */
    font-weight: 600;
    line-height: 1.5;
  }

  /* Paragraph — WCAG 1.4.12 */
  p {
    font-size: 1rem;
    line-height: 1.6;
    max-width: 65ch; /* Giới hạn độ rộng đoạn văn */
    letter-spacing: 0.01em;
  }

  /* Link — WCAG 1.4.1: không dùng màu đơn thuần, cần underline */
  a {
    text-decoration: underline;
    text-underline-offset: 3px;
    text-decoration-thickness: 1px;
  }

  a:hover {
    text-decoration-thickness: 2px;
  }

  /* Focus — WCAG 2.4.11: focus không bị che khuất */
  :focus-visible {
    outline: 3px solid #2563eb;
    outline-offset: 3px;
    border-radius: 2px;
  }

  /* Không xóa outline mà không có replacement — vi phạm WCAG 2.4.7 */
  :focus:not(:focus-visible) {
    outline: none;
  }

  /* Small / Caption text — tối thiểu 12px */
  small,
  .caption {
    font-size: 0.75rem; /* 12px — tối thiểu tuyệt đối */
    line-height: 1.5;
    letter-spacing: 0.01em;
  }

  /* Button text */
  button {
    font-family: inherit;
    font-size: 1rem;
    font-weight: 500;
    letter-spacing: 0.01em;
    /* WCAG 2.5.3: touch target tối thiểu 24x24px, khuyến nghị 44x44px */
    min-height: 44px;
    min-width: 44px;
  }

  /* Input / Form */
  input,
  textarea,
  select {
    font-family: inherit;
    font-size: 1rem; /* Tránh iOS auto-zoom (yêu cầu ≥16px) */
    line-height: 1.5;
    letter-spacing: 0.01em;
  }

  /* Label */
  label {
    font-size: 0.875rem;
    font-weight: 500;
    line-height: 1.5;
    letter-spacing: 0.01em;
  }
}

/* ── UTILITY: WCAG TEXT SPACING ──────────────────────── */
/* WCAG 1.4.12: người dùng có thể override spacing mà không mất nội dung */
@layer utilities {

  /* Dùng cho đoạn văn dài — accessibility mode */
  .text-accessible {
    line-height: 1.7 !important;
    letter-spacing: 0.12em !important;
    word-spacing: 0.16em !important;
  }

  /* Dùng cho button, label */
  .text-label {
    font-size: 0.875rem;
    font-weight: 500;
    letter-spacing: 0.02em;
    line-height: 1.5;
  }

  /* Dùng cho error message */
  .text-error {
    font-size: 0.875rem;
    font-weight: 500;
    line-height: 1.5;
    letter-spacing: 0.01em;
  }

  /* Dùng cho help text / hint */
  .text-hint {
    font-size: 0.875rem;
    font-weight: 400;
    line-height: 1.6;
    letter-spacing: 0.01em;
  }
}

/* ── MEDIA: Tôn trọng user preference ───────────────── */
/* Nếu người dùng bật "Reduce Motion" trong OS */
@media (prefers-reduced-motion: reduce) {
  html {
    scroll-behavior: auto;
  }
  *,
  *::before,
  *::after {
    animation-duration: 0.01ms !important;
    transition-duration: 0.01ms !important;
  }
}

/* Tôn trọng font-size người dùng cài trong browser */
@media (prefers-reduced-data: reduce) {
  /* Nếu băng thông thấp: dùng system font thay vì load web font */
  body {
    font-family: system-ui, -apple-system, sans-serif;
  }
}
```

## BƯỚC 5 — Component mẫu áp dụng đúng font

Tạo file `components/ui/Typography.tsx`:

```tsx
import { cn } from '@/lib/utils'
import { ReactNode } from 'react'

// H1 — Tiêu đề trang chính
export function H1({ children, className }: { children: ReactNode; className?: string }) {
  return (
    <h1 className={cn('text-4xl font-bold leading-tight tracking-tight text-gray-900', className)}>
      {children}
    </h1>
  )
}

// H2 — Tiêu đề section
export function H2({ children, className }: { children: ReactNode; className?: string }) {
  return (
    <h2 className={cn('text-3xl font-bold leading-snug tracking-tight text-gray-900', className)}>
      {children}
    </h2>
  )
}

// H3 — Tiêu đề card / nhóm
export function H3({ children, className }: { children: ReactNode; className?: string }) {
  return (
    <h3 className={cn('text-2xl font-semibold leading-snug text-gray-800', className)}>
      {children}
    </h3>
  )
}

// Body text
export function Body({ children, className }: { children: ReactNode; className?: string }) {
  return (
    <p className={cn('text-base leading-relaxed tracking-wide text-gray-700 max-w-prose', className)}>
      {children}
    </p>
  )
}

// Label cho form
export function Label({
  children,
  htmlFor,
  required,
  className,
}: {
  children: ReactNode
  htmlFor?: string
  required?: boolean
  className?: string
}) {
  return (
    <label
      htmlFor={htmlFor}
      className={cn('text-label text-gray-700', className)}
    >
      {children}
      {required && (
        <span aria-hidden="true" className="ml-1 text-red-500">*</span>
      )}
      {required && (
        <span className="sr-only">(bắt buộc)</span>
      )}
    </label>
  )
}

// Error message — screen reader sẽ đọc
export function ErrorMessage({
  id,
  children,
  className,
}: {
  id?: string
  children: ReactNode
  className?: string
}) {
  return (
    <p
      id={id}
      role="alert"
      aria-live="polite"
      className={cn('text-error text-red-600 flex items-center gap-1 mt-1', className)}
    >
      <span aria-hidden="true">⚠</span>
      {children}
    </p>
  )
}

// Hint / Help text
export function HintText({ id, children, className }: { id?: string; children: ReactNode; className?: string }) {
  return (
    <p id={id} className={cn('text-hint text-gray-500 mt-1', className)}>
      {children}
    </p>
  )
}

// Caption / small text
export function Caption({ children, className }: { children: ReactNode; className?: string }) {
  return (
    <small className={cn('text-xs leading-normal tracking-wide text-gray-500', className)}>
      {children}
    </small>
  )
}
```

## CHECKLIST KIỂM TRA SAU KHI CẤU HÌNH

Sau khi agent hoàn thành, kiểm tra các điểm sau:

- [ ] `<html lang="vi">` có mặt trên tất cả trang
- [ ] Body text không nhỏ hơn `16px` (1rem)
- [ ] Phóng to trình duyệt lên 200%: layout không bị vỡ, chữ vẫn đọc được
- [ ] Không có `font-size` nào dùng đơn vị `px` cứng cho text (phải dùng `rem`)
- [ ] Không có `line-height` nào nhỏ hơn `1.5` cho đoạn văn
- [ ] Focus indicator hiển thị rõ khi dùng Tab điều hướng
- [ ] Không có `outline: none` không có replacement
- [ ] Tiếng Việt có dấu hiển thị đúng, không bị thiếu ký tự
- [ ] Button và input tối thiểu `44px` chiều cao
- [ ] Không dùng `font-weight: 300` (quá mỏng cho low vision)
- [ ] `prefers-reduced-motion` được xử lý trong CSS

## LÝ DO KHÔNG DÙNG TIMES NEW ROMAN

> **Times New Roman và các Serif font (có chân) được thiết kế cho in ấn trên giấy từ năm 1931.**
> Trên màn hình số, các chân chữ nhỏ bị pixel hóa, gây mờ và mỏi mắt — đặc biệt nguy hiểm
> với người low vision phải đọc ở kích thước phóng to 200–400%.
> 
> **Quy tắc:** Giấy in → Serif | Màn hình số → Sans-serif.
